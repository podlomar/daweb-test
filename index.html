13.

Uvažte následující React komponentu:

import React, { useState } from 'react';

export const LightSwitch = () => {
  const [isOn, setIsOn] = useState(true);

  return (
    <button
      className="btn"
      onClick={() => setIsOn(!isOn)}
    >
      Zhasnout
    </button>
  );
};

LightSwitch při každém kliknutí na tlačítko mění hodnotu stavu isOn z true na false a obráceně.

Chceme, aby se tato změna projevila vizuálně i na tlačítku. Předpokládejme, že v CSS máme nastylovanou třídu btn⁠-⁠-⁠on. Když tuto třídu přidáme na tlačítko, bude tlačítko zvýrazněné jako zapnuté. Třídu btn mějme na tlačítku vždy.

Zároveň chceme, aby text tlačíka byl Zhasnout pouze v případě, že isOn je true. V opačném případě nastavte tlačítku text na Rozsvítit.

Upravte aplikaci uvedenou výše tak, aby se na tlačítku objevila CSS třída btn⁠-⁠-⁠on, je-li stav isOn nastaven na true. Je-li ve stavu isOn hodnota false, třída na tlačítku nebude.

Jako řešení pošlete kód komponenty LightSwitch.
