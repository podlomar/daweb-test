14.

Vypínač světla

Uvažte následující React komponentu:

import React, { useState } from 'react';

export const LightSwitch = () => {
  const [isOn, setIsOn] = useState(true);

  return (
    <button
      className="btn"
      onClick={() => setIsOn(!isOn)}
    >
      Zhasnout
    </button>
  );
};

LightSwitch při každém kliknutí na tlačítko mění hodnotu stavu isOn z true na false a obráceně.

Chceme, aby se tato změna projevila vizuálně i na tlačítku. Předpokládejme, že v CSS máme nastylovanou třídu btn⁠-⁠-⁠on. Když tuto třídu přidáme na tlačítko, bude tlačítko zvýrazněné jako zapnuté. Třídu btn mějme na tlačítku vždy.

Zároveň chceme, aby text tlačíka byl Zhasnout pouze v případě, že isOn je true. V opačném případě nastavte tlačítku text na Rozsvítit.

Upravte aplikaci uvedenou výše tak, aby se na tlačítku objevila CSS třída btn⁠-⁠-⁠on, je-li stav isOn nastaven na true. Je-li ve stavu isOn hodnota false, třída na tlačítku nebude.

Jako řešení pošlete kód komponenty LightSwitch.

Vtip

Prohlédněte si vzorovou webovou aplikaci Dad Jokes napsanou pomocí čistého JavaScriptu. Aplikace zobrazuje vtip s avatarem uživatele a dvěma tlačítky like a dislike. Vyzkoušejte si, jak tlačítka fungují.

Vaším úkolem bude vytvořit tutéž aplikaci v Reactu.

  1. Celý kód vzorové aplikace v HTML, CSS a JS najdete v tomto repozitáři. Na GitHubu si prohlédněte kód aplikace. Toto repository nebudete potřebovat na svém počítači, stačí používat náled na GitHubu.
  2. Pracovat budete v jiném repository. Vytvořte si kopii repository ze šablony projekt-dadjokes do svého GitHub účtu. Repozitář si naklonujte do počítače a vytvořenou složku si otevřete ve VS Code.
  3. Ve VS Code otevřete terminál, nainstalujte závislosti a spusťte vývojový server: 
    npm install
npm run dev
  4. Naprogramujte vaši Reactovou aplikaci tak, aby dělala přesně totéž a vypadala přesně stejně, jako vzorová aplikace. Stačí mít jednu komponentu HomePage, zatím nemusíte vytvářet žádné další. Ze vzorové aplikace můžete vykrást libovolný kód, který se vám bude v Reactu hodit. Obrázky pro like a dislike už máte připravené ve složce /img, přesuňte je na právné místo do komponenty HomePage.
  5. Jako odpověď na tuto otázku uveďte odkaz na váš GitHub repozitář (adresu zkopírujte z adresního řádku prohlížeče).

Pokračování vtipu

Pokračujte v projektu z předchozí otázky.

  1. Vytvořte komponentu Joke, která zobrazuje jeden vtip. Tato komponenta bude sídlit ve vlastní složce se všemi potřebnými CSS styly a obrázky. Komponenta Joke nechť má pět props: userAvatar, userName, text, likes, dislikes.
  2. Pomocí jsonhost vytvořte API endpoint, který bude vracet data z JSON souboru jokes.json. Zobrazte každý vtip z tohoto enpointu pomocí komponenty Joke.
  3. Zvažte, která vlastnost ze zobrazovaných dat se nejlépe hodí pro key prop vaší komponenty a použijte ji jako key.

Výsledná aplikace by měla zobrazovat seznam všech vtipů se správnými uživateli a počty likeů. Tlačítka u každého vtipu by měla fungovat tak, že mění počty likeů uvnitř každé jednotlivé komponenty.
